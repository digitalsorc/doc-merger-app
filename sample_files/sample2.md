# Sample Document 2: Configuration Guide

Learn how to configure the Markdown Merger application.

## Basic Settings

These settings are available in the main window:

- **Preset**: Choose from pre-configured settings
- **Separator**: Visual divider between documents
- **Header**: How documents are labeled

## Advanced Settings

For more control, use the Advanced Settings dialog.

### Document Structure

| Option | Description |
|--------|-------------|
| Header Template | Template for section headers |
| Include Path | Show source file path |
| Separator Style | Custom separator markdown |

### AI Optimization

Enable these for knowledge base use:

1. Document Metadata
2. Semantic Markers
3. Chunk Hints

## Tips

> **Pro Tip**: Use the "ai_knowledge_base" preset for best results with RAG systems.

## Summary

Configuration is flexible and can be saved for reuse.
